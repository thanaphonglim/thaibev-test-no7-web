# thaibev-product-test-web

Frontend สำหรับระบบจัดการสินค้า พัฒนาโดยใช้ Angular

## Tech Stack
- Angular 21.0.3
- Bootstrap 5
- Angular Material 21.2.1
- "angularx-qrcode": "^21.0.4"

## Features
- แสดงรายการสินค้า
- เพิ่มสินค้า
- ลบสินค้า
- แสดง Qrcode

## environment 
- src/environments/environment.ts

## Run
- npm install
- ng serve
